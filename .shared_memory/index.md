# OpenMailStack Shared Memory Index

Last reviewed: 2026-06-21.

Use this folder for repo-specific implementation state, commands, risks, and task lessons. Do not store secrets, passwords, raw credential logs, or mailbox contents.

## Files

- `implementation_state.md`: current architecture, deployment shape, frontend/backend behavior, and validation inventory.
- `change_log.md`: dated summaries of meaningful code, deployment, and verification changes.
- `commands.md`: repeatable local commands and validation entry points.
- `risk_register.md`: known risks, gaps, and follow-up work.
